# Administrador de Productos (Front-end)

Aplicación web para la gestión de productos, construida con React, TypeScript y Vite.

## Stack

- React 19
- TypeScript
- Vite
- React Router
- React Hook Form
- Tailwind CSS
- Axios
- Valibot

## Requisitos

- Node.js 20+
- pnpm

## Instalación

```bash
pnpm install
```

## Scripts disponibles

- `pnpm dev`: inicia el entorno de desarrollo
- `pnpm build`: genera el build de producción
- `pnpm preview`: previsualiza el build generado
- `pnpm lint`: ejecuta ESLint

## Deploy

Completa esta sección cuando publiques el proyecto.

- URL de producción: `PENDIENTE`

## Variables de entorno

Si necesitas configurar endpoints o llaves, crea un archivo `.env` en la raíz del proyecto.

Ejemplo:

```env
VITE_API_URL=http://localhost:4000
```
